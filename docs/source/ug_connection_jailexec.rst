connection vbotka.freebsd.jailexec
----------------------------------

The `connection plugin vbotka.freebsd.jailexec`_ connects the jails without SSH. The utility
``jexec`` at the jail host is used to run the commands within the jails.

.. seealso::
   * `man jexec`_
